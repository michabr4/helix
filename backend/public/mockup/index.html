Embed flow (application)

End-to-end sequence implemented in this repository.

  1. User signs in on Operations (/) — JWT stored.
  2. Browser loads Power BI page (/powerbi-pm.html) — fetches embed context with Authorization: Bearer.
  3. Backend uses service principal to call Power BI GetReport + GenerateToken.
  4. powerbi.embed renders the report inside #embedContainer (token type Embed).

React shell can call the same API and embed via powerbi-client (see frontend route /powerbi-pm).

Digital Document Solutions (DDS)

Digitized delivery document management — automated generation, secure sharing, version control, and approval workflows for customer-facing technical documents tied to service delivery milestones.

Document lifecycle

Generate
Template-driven document creation from project data, device inventory, and milestone context. API-triggered or manual.
Review & Approve
Multi-stakeholder approval workflow. Role-gated access for SDM, PM, CXM, and customer reviewers. Audit trail for compliance.
Share & Distribute
Secure portal links with expiry. Customer-branded packages. Bulk distribution for property portfolios. Download and print support.
Archive & Version
Full version history. Diff comparison between revisions. Retention policy enforcement. Tied to project lifecycle in SDM.
